# Prime Tetration Trading Application

A web application for generating price projections using advanced mathematical models based on prime tetration towers and crystalline lattice oscillators.

## Project Structure

This project follows a clean, organized structure:

- **`/backend`** - Express server and API endpoints
- **`/frontend`** - Client-side code (HTML, CSS, JavaScript)
- **`/docs`** - Documentation and guides
- **`/scripts`** - Utility and deployment scripts

See [docs/PROJECT_STRUCTURE.md](docs/PROJECT_STRUCTURE.md) for detailed structure information.

## Quick Start

```bash
# Install dependencies
npm install

# Start the server
npm start
# or
./scripts/start.sh

# Open http://localhost:8080/index.html in your browser
```

## Building CSS

```bash
npm run build:css
```

## Features

- Real-time stock data integration via Finnhub API
- Prime tetration tower projections
- 12-dimensional crystalline lattice oscillators
- Mobile-responsive design with Android optimization
- Interactive charts with zoom and pan capabilities

## Technology Stack

- **Backend**: Node.js 18+ with Express.js
- **Frontend**: Vanilla JavaScript, Chart.js, Tailwind CSS
- **API**: Finnhub for stock data
